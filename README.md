# Arduino Assignment 1: Smart Countdown
## Author
**NAME**: Nuru Ahmed Fawaz Napari
**COURSE**: Programming C++ for Engineers
## PROJECT DESCRIPTION
This project demonstrates the fundamentals of Arduino programming by implementing a smart countdown system. The program counts down from 5 to 1 using a while loop. During each countdown step, the built-in LED flashes the same number of times as the current count, while the value is displayed on the serial monitor.

## Concepts demonstrated:
1. Variables (integers)
2. Functions with parameters
3. Delay
4. Serial and Serial.print()

## Hardware used:
1. Arduino Uno
2. USB cable

## Instructions:
1. Open the project in Arduino IDE.
2. Connect the Arduino Uno via USB.
3. Select the board (Arduino Uno) and the correct COM port.
4. Upload the sketch.
5. Open the serial monitor and set the baud rate to 9600.
